  1. Que se passe-t-il lorsque l'on visualise la fenêtre exo1.html ?

    un temps-mort d'une seconde est lancé et le message "tick" s'affiche après dans la console.

  2. Expliquez ce que fait window.onload = loaded puis window.setTimeout(refresh, 1000) ?

    window.onload est un gestionnaire d'évènement qui se déclenche quand toute les ressources de la page on fini de charger.
    window.setTimeout(refresh,1000) est une méthode globale permettant de définir un minuteur qui exécute une fonction ou un code donné après la fin du délai indiqué.
